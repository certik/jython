JPython Documentation

JPython is a new implementation of the Python language.  Every effort is being made to ensure that the Java and C implementations of Python will be as compatible as possible.  As a result, you should be able to use the existing Python documentation for most purposes.  When things begin to stablize, a detailed listing of differences from the Python documentation will be provided.

The following documentation is specific to JPython:

  • Invoking the JPython interpreter - type "jpython" at a shell prompt
  • The JPython Registry - replaces environment variables for PYTHONPATH and more
  • Embedding JPython - how to call JPython from a Java application
  • Interaction with Java Packages
    • The Basics
    • JavaBean Properties - makes all Java classes easier to use from Python
    • jarray - Building and using Java arrays from Python
    • Subclassing Java classes from Python
    • Compiling Python Source to Real Java Classes - the jpythonc utility - building applets, servlets, beans, ...
  • Compiling JPython - how to use the files in src
